画像の表題

image-captions プラグインを使うと，画像のタイトルを追加することが出来ます．

{
    "plugins": ["image-captions"]
}

標準では画像の右下の位置に，接頭辞 Figure を alt 内容に追加したものが出力されるようです． これはプラグイン設定で変更することができます．例えば，次のように設定すると

{
    "pluginsConfig": {
        "image-captions": {
            "caption": "Fig. _BOOK_IMAGE_NUMBER_ _CAPTION_",
            "align": "left"
        }
    }
}

のようになります．細かく調整できますので，詳細は以下を参照してください．

  • gitbook-plugin-image-captions
