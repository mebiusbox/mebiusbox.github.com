Root ノード

エフェクトに１つだけ存在する特別なノード。この Root ノードを起点にツリーを構築していきます。

Null ノード

特に何もしないノード。寿命管理などに使用します。

Emitter ノード

出力ポート「particle」に接続したノードツリーを生成するノード。

Particle ノード

パーティクルを表示するノード。

Stripe ノード

連続した頂点を生成するノード。生成した頂点をつなげてポリゴンを表示します。
出力ポート「modifier」は生成した頂点に対して処理するモディファイアノードを接続します。

Transform ノード

変形ノードです。モディファイアノードをまとめるためのものです。
出力ポート「modifier」にモディファイアノードを接続します。
modifier に接続したノードは上から順番に処理されます。通常モディファイアの影響は下から上ですが、その逆になっていますので注意してください。

Translate ノード

位置に変化をつけるモディファイアノード。

Rotation ノード

回転に変化をつけるモディファイアノード。

Scale ノード

スケールに変化をつけるモディファイアノード。

TRS ノード

位置、回転、スケールに変化をつけるモディファイアノード。
演算順番は スケール→回転→位置 です。

Size ノード

サイズに変化をつけるモディファイアノード。

Color ノード

カラーに変化をつけるモディファイアノード。

Color ノード

カラーに変化をつけるモディファイアノード。

Gravity ノード

重力をかけるフィールドノード。

Wind ノード

風力を適用するフィールドノード。

Radial ノード

放射状に他のノードに影響を与えるフィールドノード。
