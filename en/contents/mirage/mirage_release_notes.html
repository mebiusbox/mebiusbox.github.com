リリースノート

1.0.2.0

  • Fixed : Scene : 再生中にシーンウィンドウのレイアウトを変更すると再生が停止してしまう不具合を修正しました。
  • Fixed : Animation Editor : 更新ボタンを押すと強制終了する不具合を修正しました。
  • Fixed : Animation Editor : アニメーションを選択していない状態でフレーミングをするとフリーズする不具合を修正しました。
  • Misc : Manual : テクスチャパネルの UV の説明を更新しました。
  • Misc : Manual : テクスチャパターンエディタに「パターンの保存」を追加しました。

1.0.1.0

  • 前バージョンとの保存データの互換性がなくなりました。
  • その他いろいろ。

1.0.0.15

  • ノードの名前をスケマティック画面に表示するようにしました。
  • Fixed: インスペクタウィンドウの「Color」において、カラー選択画面から色を選択すると不正な値が設定される不具合を修正しました。

1.0.0.14

  • 前バージョンとの保存データの互換性がなくなりました。
  • ランダム上限値やキーフレームを設定できるパラメータの入力部分で、値を３つから２つに変更しました。一番左がなくなり、キーフレームを設定した場合は初期値で設定されます。
  • ノードを矩形選択する際に、矩形に交差したものを選択するようにしました。（いままではノードが完全に矩形内に含まれていた場合）
  • ノードの矩形選択をスムーズにしました。
  • ノードの「活動開始」「寿命」にランダム値を設定できるようにしました。
  • シーンウィンドウにおいて、X軸(赤)、Z軸(青)を表示するようにしました。
  • Fixed: コピー＆ペーストが出来ない不具合を修正しました。
  • Fixed: モディファイア「Translate」「Rotate」「Scale」「Color」を複数つなげても、最後の１つしか反映されていなかった不具合を修正しました。
  • Fixed: ノードを選択したときに、インスペクタに不正な値が設定される場合がある不具合を修正しました。
  • Fixed: 「Emitter」の生成間隔で、「Per Second」以外が機能していなかった不具合を修正しました。

1.0.0.13

  • Fixed: 保存できない不具合を修正しました。

1.0.0.12

  • 高速化しました。

1.0.0.11

  • フィールド「Gravity」「Wind」「Radial」が有効になりました。

1.0.0.10

  • インスペクタウィンドウにおいて、パネルの開閉情報を各ノードに保存するようにしました。（ファイルにも保存されます）
  • 属性「基本」の「表示」が有効になりました。
  • デスクトップにショートカットを作成するようにしました。

1.0.0.8

  • 前バージョンとの保存データの互換性がなくなりました。
  • ノード接続（親子）において、ポートではなく、ノード自体の領域にドラッグして接続できるようになりました。
  • ポートから何もないところにドラッグすると、作成するノードメニューが表示されノードを選択すると、選択したノードを作成し、かつドラッグ元のノードと接続された状態になるようにしました。
  • インスペクタウィンドウの Color, Color2 を変更しました。
  • 初期値、ランダム上限、ランダム下限を廃止し、入力箇所を２つに絞りました。詳しくはマニュアル「属性」の先頭部分を参照してください。
  • 回転の値を弧度から角度に変更しました。

1.0.0.5

  • まともに動かなかった不具合を修正しました。

1.0.0.4

  • メインフォームにツールバーを追加しました。
  • テクスチャ画像選択画面で、フォルダの履歴を保存するようにしました。
  • タイトルバーに現在編集中のファイル名を表示するようにしました。
  • Fixed: アニメーションエディタ上でキーフレームを削除すると強制終了する不具合を修正しました。
  • Fixed: Color, Translate, Rotate, Scale ノードを Root や Emitter などに直接接続すると強制終了する不具合を修正しました。

1.0.0.2

  • マウスによるカメラ操作を変更しました。
  • テクスチャ画像選択画面を変更しました。
  • インスペクタウィンドウのテクスチャパネルにおいて、テクスチャプレビューを表示するようにしました。
  • Fixed: 活動開始時間以前でも描画されていた不具合を修正しました。
  • Fixed: パーティクルに寿命を設定した状態で再生すると、強制終了する不具合を修正しました。
  • Fixed: プレイコントロールにおいて、先頭にシークしたとき、再描画されない不具合を修正しました。
  • Fixed: ループ再生しているときに、シーンが初期化されない不具合を修正しました。
