開発環境

Microsoft Visual Studio 2008 SP1


InnoSetup

インストーラは InnoSetup を使用して作成しています。InnoSetup はインストーラ作成用のアプリケーションです。以下からダウンロードできます。

http://www.jrsoftware.org/isinfo.php

バージョンは 5.3.1.0 の Unicode 版を使用しています。また、日本語ファイル(japanease.isl)が必要です。これは Translations から入手できます。

./InnoSetup/Mirage.iss を InnoSetup で読み込み、AppVerName と VersionInfoVersion を更新します。後は、Compile をすればセットアップファイルが作成されます。


再配布ファイル

再配布に必要なファイルが ./redist にあります。ライセンス文書や実行に必要な DLL などが入っています。


ライブラリ

./lib に必要なライブラリが入っています。また、SlimDX SDKが必要です。


ライブラリソース

./lib_src にはライブラリのソースファイルが入っています。これは、既存のライブラリに変更を加えたバージョンです。


XML ファイル

KeyMap.xml:キーマップ情報。コマンドとキー情報が入っています。
Command.xml:コマンドのスコープ情報と、説明が入っています。
