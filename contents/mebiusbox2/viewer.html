Mebiusbox 2.x ユーザーガイド

ビューア

Mebiusbox2 では以下のビューアを内蔵しています。

  • テキスト
  • バイナリ
  • 画像
  • PDF
  • WindowsMediaPlayer
  • ウェブブラウザ

テキストビューア

 



テキストを表示します。簡単なハイライト機能もあります。

バイナリビューア

 



１６進数で表示します。

画像ビューア

 



画像を表示します。

PDF ビューア

 



PDF ファイルを表示します。

別途 AcrobatReader をインストールする必要があります。

Windows Media Player

 



Windows Media Player で再生します。

ウェブブラウザ

 



ウェブブラウザで表示します。

ビューアの表示

ビューア表示ボタンを押します。


ビューアの表示位置

デフォルトでは右側ですが、オプションで下部に変更することができます。
