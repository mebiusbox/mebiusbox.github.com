Mebiusbox 2.x ユーザーガイド

ビュー

Mebiusbox2 ではフォルダ内のコンテンツを表示する多様なビューがあります。

  • Details
  • Thumbnails
  • Html
  • Mix
  • Movie
  • SmileVideo

Details : 詳細

 



アイテムの情報を詳細に表示します。

Thumbnails : サムネイル

 



画像（動画）のサムネイルを表示します。

Html : Web

 



ウェブブラウザで表示します。

Mix : ミックス

 



画像（動画）のサムネイルと、アイテムの詳細を表示します。

Movie : 動画

 



動画（画像）のサムネイルを複数表示します。

SmileVideo : ニコニコ動画ランキング

 



ニコニコ動画ランキングを表示します。

ビューの切替

ビュー切替ボタンを押して、切り替えたいビューを選択します。


ビューのロック

フォルダを切り替えると、ビューも自動的に切り替わります。
しかし、場合によってはこれが望まないことかもしれません。
その場合に、ビューをロックすることでビューの自動切換えを無効にすることができます。
ビューをロックするには、ビュー切替ボタンを押して、「ロック」を選択します。
