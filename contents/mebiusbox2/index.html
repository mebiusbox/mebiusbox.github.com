Mebiusbox 2.x

Mebiusbox2 はファイル整理＋αなソフトです。
FTP フォルダ や 書庫ファイル内をブラウズすることができ、また、Everything(*1)を 使ってデスクトップ検索をすることが出来ます。さらに、仮想フォルダ機能、Web、 フィード、ニコニコ動画のランキングも表示することが可能です。
その他には、強力なフィルター機能、タグ機能、リネーム機能、ファイル（フォルダ）の 連番作成機能などが備わっています。また、内蔵ビューアとして、テキスト、バイナリ、 Web, PDF, 画像、Windows Media Player があります。

本ソフトウェアはフリーソフトウェアです。

動作環境

Windows XP(32bit), Windows Vista(32bit) に対応しています。Windows7 は RMT版で一応動作確認しています。

Mebiusbox2 を動作させるには「Microsoft Visual C++ 2008 ランタイムライブラリ」が必要です。
インストーラ版では、同時にインストールされます。アーカイブ版をご利用の方は別途インストールしてください。 ランタイムライブラリは以下の場所から入手できます。

→http://www.microsoft.com/Downloads/details.aspx?displaylang=ja&FamilyID=a5c84275-3b97-4ab7-a40d-3802b2af5fc2
