ページのトップに戻るボタン

navigator プラグインでも追加することは出来ますが，外観がよい back-to-top-button プラグインもオススメです．このプラグインはデフォルトテーマの各テーマカラーに対応しています．

{
    "plugins": ["back-to-top-button"]
}
