UML

uml プラグインを使います．

{
    "plugins": "uml"
}

次のように使います．

{% uml %} @startuml

Class Stage
Class Timeout {
    +constructor:function(cfg)
    +timeout:function(ctx)
    +overdue:function(ctx)
    +stage: Stage
}
 Stage <|-- Timeout

@enduml {% enduml %}
