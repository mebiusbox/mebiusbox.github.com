# Three.js

## pixy.js

Three.js 用の独自シェーダとユーティリティライブラリです．

### 機能

- 物理ベースレンダリング（拡散反射：ランバート，オーレン・ナイヤー，鏡面反射：クック・トランス）
- ディファードシェーディング（three.js の MRT カスタムバージョンを使用）
