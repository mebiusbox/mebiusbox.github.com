DiskCatalogManager 2.7.4

DiskCatalogManagerは、ハードディスクやＣＤ－ＲＯＭ，ＤＶＤ－ＲＯＭ、ＵＳＢメモリなどの メディア に記録されているファイルやフォルダの情報を カタログ というデータに保存して管理するソフトウェアです。 カタログを作成することで、あるファイルやフォルダがどのメディアに存在するかを、簡単に検索することができます。検索するときはメディアは不要ですので、ＣＤ－ＲＯＭの取替えやＵＳＢメモリなどの取り外しといった手間がかかりません。

カテゴリ機能​

作成したカタログをカテゴリに振り分けることが出来ます。 カテゴリを利用すると、カタログの管理がしやすくなり、検索を高速化したりすることができます。

書庫ファイルに対応​

別途ＤＬＬにより、書庫ファイルに含まれているファイルやフォルダの情報もカタログに加えることができます。

  • UNLHA32.DLL
  • UNZIP32.DLL
  • CAB32.DLL
  • TAR32.DLL
  • UNGCA32.DLL
  • UNRAR32.DLL
  • YZ1.DLL
  • 7-zip32.DLL

サムネイル作成​

画像のサムネイルや動画のサムネイルを作成し、カタログに加えることができます。 画像のサムネイルは標準でサポートしている画像形式以外にも Susie Plugin を導入することで様々な画像形式に対応することも出来ます。 動画のサムネイル作成にはその動画形式に対応した DirectShowフィルタ が必要です。
